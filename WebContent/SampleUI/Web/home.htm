JQuery

JQuery is a javascript library, which has wide range of actions such as Event handling, animation, html document traversing and Ajax interaction for web development. JQuery simplifies javascript programming.

Simple JQuery Example

JQuery provides a simple way to select single element or group of elements. You can access element by type (div, span, p), id, CSS class and attribute etc. In this example, I am going to register the click event on the all DIVs and on the click I will assign CSS class to selected div element.

Second Div
Third Div 

                    
    $('div').click(
        function () {
            $('.SelectDiv').removeClass('SelectDiv');
            $(this).addClass('SelectDiv');
        }
    );

$(‘div’) will selected all Div element in the page and $(‘div’).click(function(){….}); means the on the click of div the function inside the click() will call.

As I mentioned above that you can select elements by different ways using JQuery. $('.SelectDiv') Statement will select html elements which has CSS class “SelectedDiv”. removeClass('SelectDiv'); removes the Css class “SelectDiv” from all selected elements and $(this).addClass('SelectDiv'); adds/assigns the Css class to current clicked element.
